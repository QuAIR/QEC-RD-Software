# API Reference

This section documents the Stage 1 Python API that users and contributors are expected to work with.

## Scope

The preferred public surface is:

- `qec_rd.api` for user-facing experiment entry points
- `qec_rd.core` for shared platform objects and result containers
- `qec_rd.kernel.*` for implementation modules that back the Stage 1 flow

## Start Here

If you are new to the repository, begin with:

1. [Public API](public-api.md)
2. [Core Models](core.md)
3. [Kernel Modules](kernel.md)

The API reference complements the end-to-end demos. The demos show how to use the system; the API pages show what each exported object and function is.
